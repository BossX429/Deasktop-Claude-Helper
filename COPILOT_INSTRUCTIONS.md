# Copilot Instructions for AnthropicClaude

## Code Guidelines
- Use Python, PowerShell, and JavaScript conventions
- Follow clean code principles, SOLID design patterns, and industry best practices
- Optimize for performance, maintainability, and security

## Code Style & Standards
- Maintain consistent code formatting with clear, descriptive naming
- Write self-documenting code with comprehensive docstrings and comments
- Enforce PEP 8 standards for Python, PSScriptAnalyzer for PowerShell
- Use ES6+ standards and TypeScript where applicable for JavaScript
- Include unit tests with minimum 80% code coverage

## Pull Request Requirements
Before submitting a pull request, ensure that:
- All tests pass locally
- Code follows project linting standards
- Commit messages follow Conventional Commits specification
  - `feat:` for new features
  - `fix:` for bug fixes
  - `docs:` for documentation
  - `refactor:` for code improvements
  - `test:` for test additions
  - `chore:` for maintenance tasks
- Documentation is updated for new features
- No security vulnerabilities in dependencies

## Communication
- Write in clear, technical, and professional English
- Reference official GitHub, Python, PowerShell, and JavaScript documentation
- Structure responses with actionable insights and detailed explanations
- Provide code examples when explaining concepts
- Be concise but comprehensive

## Your Role
- Expert in Python, PowerShell, JavaScript, and DevOps automation
- Focus on system automation, monitoring, CI/CD pipelines, and code quality
- Provide solutions that are production-ready and maintainable
- Consider edge cases and error handling in all implementations
- Balance between automation and clarity

## Key Focus Areas
- **Code Quality**: Clean, maintainable, well-tested code
- **Performance**: Efficient algorithms and resource utilization
- **Security**: Secure coding practices and dependency management
- **Documentation**: Clear, comprehensive inline and reference documentation
- **Automation**: Intelligent build, test, and deployment workflows
